연결리스트(LinkedList)란?

C언어를 공부할 때 배열을 배웠을 것이다. 배열과 마찬가지로 연결리스트는 변수들을 묶어서 사용할 수 있는 자료형이다. 배열은 변수를 순차적으로 저장하는데 반해 연결리스트는 변수끼리 포인터로 연결하여 저장한다는 것이 차이점이다.


배열과 연결리스트의 구조(연결리스트에서는 원소를 노드라 한다.)


배열이 있는데 왜 굳이 연결리스트를 사용하는걸까? 배열은 인덱스를 이용하여 원소에 바로 접근할 수 있다는 장점이 있지만 원소를 첫번째 또는 중간에 추가/삭제할 때에는 뒤에 원소들의 인덱스를 다 이동시켜야 하는 불편함이 있다. 하지만 연결리스트는 어디에 추가/삭제하든 앞의 노드와 추가할 노드의 포인터 만 변경하면 되기 때문에 추가/삭제에 용이하다.

하지만 연결리스트는 탐색에 불리하다. 배열은 오름차순이든 내림차순이든 정렬이 되어있다면 인덱스를 이용하여 바로 원소에 접근이 가능하지만 연결리스트는 첫번째 노드부터 마지막 노드까지 지나가며 탐색해야 하기 때문이다.

필자는 연결리스트에서 노드를 추가/삭제하려면 일단 위치를 탐색해야 하는데 그러면 추가/삭제도 배열에 비해 좋다고 할 수 있나?라고 생각했는데 노드를 첫번째 위치에 추가/삭제 할 경우에는 탐색할 필요가 없기 때문에 이점이 있다. 그리고 후에 이중연결리스트, 큐/스택 등에서도 연결리스트를 이용하여 구현하기 때문에 익혀둘 필요가 있다.

#include<stdio.h>
#include<stdlib.h>
 
typedef struct node{
    int data;
    struct node *next;
}node;
 
void insertNode(node *h, int data){
    node *newNode = (node*)malloc(sizeof(node));
    newNode->data = data;
 
    if(h->next == NULL){
        newNode->next = h->next;
        h->next = newNode;
 
        return;
    }
 
    node *temp = h->next;
 
    while(temp->next != NULL){
        temp = temp->next;
    }
 
    newNode->next = temp->next;
    temp->next    = newNode;
}
 
void deleteNode(node *h){
    if(h->next == NULL){
        printf("List is Empty!\n");
        return;
    }
 
    node *pre;
    node *temp = h->next;
 
    while(temp->next != NULL){
        pre  = temp;
        temp = temp->next;
    }
 
    pre->next = NULL;
    free(temp);
}
 
void printList(node *h){
    if(h->next == NULL){
        printf("List is Empty!\n");
        return;
    }
 
    node *temp = h->next;
 
    printf("List = [ ");
 
    while(temp != NULL){
        printf("%d ", temp->data);
 
        temp = temp->next;
    }
 
    printf("]\n");
}
 
int main(){
    node *h = (node*)malloc(sizeof(node));
    h->next = NULL;
 
    insertNode(h, 25);
    insertNode(h, 30);
    insertNode(h, 35);
    insertNode(h, 40);
    printList(h);
}

1. insertNode

insertNode는 리스트의 마지막에 노드를 추가하는 함수이다. 연결리스트에 노드를 추가하는 알고리즘은 아래와 같이 간단하다.

1. 새로운 노드를 생성한다.
2. 추가할 위치의 전 노드의 next값을 받는다.
3. 전 노드의 next값을 새 노드의 주소값으로 변경한다.

여기서 신경쓸 것은 추가할 위치를 구하는 것인데, 중간에 노드를 추가할 거라면 while문에서 조건을 (temp->data != 전 노드의 data)와 같이 조금만 바꿔주면 쉽게 해결 할 수 있다. 물론 while문 내부에 next이 NULL인 노드를 만날 때 조건을 처리해줘야 한다.

2. deleteNode

마지막 노드를 삭제하는 함수로 연결리스트에서 노드를 삭제하는 알고리즘은 다음과 같다.

1. 삭제할 노드의 next값을 전 노드의 next값에 넘겨준다.
2. free()함수를 사용하여 삭제할 노드의 메모리를 해제한다.

노드를 추가할 때와 마찬가지로 위치만 탐색하여 위의 알고리즘대로 구현하면 된다. 위의 코드에서는 pre변수를 사용하여 반복문 내에서 이동하기 전의 temp변수를 저장해놓으면서 이전 노드를 구했다.

3. printList

리스트 내의 노드를 출력하는 방법은 첫 노드부터 탐색하면서 하나하나 printf()를 사용하여 출력해주면 된다. 간단하다.
